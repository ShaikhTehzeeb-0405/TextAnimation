# TextAnimation Project

This is a mini-project demonstrating text animation effects using HTML and CSS.

## Features
- A creative and colorful text animation that showcases dynamic text effects.
- The animation runs smoothly and can be customized further.
- Simple and lightweight design for easy integration.

## Technologies Used
- **HTML**: Structure of the page.
- **CSS**: Styling and animation effects.
